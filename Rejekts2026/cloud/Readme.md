# Cloud demos

For these demos you'll need accounts on Hetzner and Akamai.
Cost is negligible; you can leave the demo cluster running for hours for a few cents of cost.

* [`hetzner/`](hetzner/Readme.md) has instructions and scripts for provisioning on Hetzner.
* [`akamai/`](akamai/Readme.md) has instructions and scripts for provisioning on Akamai.
